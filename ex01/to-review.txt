STL (Standard Template Library)

1. Containers ⭐⭐⭐⭐⭐ (Most Important)

Know what a container is.

A container is a class that stores a collection of objects.

Examples:

std::vector<int>
std::list<int>
std::deque<int>
std::stack<int>

Know the differences:

Container	Use
vector	Dynamic array
deque	Double-ended array
list	Linked list
stack	LIFO stack

For CPP08:

ex00 → vector, list, deque
ex01 → vector
ex02 → stack
2. Iterators ⭐⭐⭐⭐⭐

This is probably the biggest new concept.

Instead of pointers or indices, STL uses iterators.

std::vector<int>::iterator it;

Example

std::vector<int> v;

v.push_back(10);
v.push_back(20);

std::vector<int>::iterator it = v.begin();

std::cout << *it; // 10

++it;

std::cout << *it; // 20

Know:

begin()
end()
++it
*it
3. STL Algorithms ⭐⭐⭐⭐⭐

Instead of writing loops yourself, use algorithms.

Learn:

std::find()
std::sort()
std::min_element()
std::max_element()
std::distance()

Example

std::find(v.begin(), v.end(), 42);

CPP08 is basically:

Containers + Iterators + Algorithms

8. Time Complexity ⭐⭐☆☆☆

You don't need to memorize everything, but understand why different algorithms are chosen.

Example:

Operation	Complexity
push_back()	O(1)
sort()	O(n log n)
find()	O(n)

This helps explain why Span::shortestSpan() sorts first.

CPP Module 08

STL Containers • Iterators • Algorithms • Generic Programming

CPP08 introduces the Standard Template Library, especially containers and iterators. Instead of manually managing every collection, you start using tools such as std::vector, std::list, std::stack, and standard algorithms.

Each exercise focuses on a different STL concept:

ex00: Find a value inside a container using iterators and templates.
ex01: Create a Span class that stores integers and calculates the shortest and longest distances between them.
ex02: Extend std::stack into an iterable container called MutantStack.


Concepts by Exercise
Exercise	Concepts Introduced
ex00	STL Containers, Function Templates, Iterators, std::find, Exceptions
ex01	std::vector, Algorithms, Sorting, Range Insertion, Shortest and Longest Span
ex02	Container Adaptors, Inheritance, Iterators, std::stack, Underlying Containers
